# MPVM – Mot de Passe Visuel Multidimensionnel  
Prototype officiel – Bruno Valls-Gabaud

Ce dépôt contient le prototype visuel du **MPVM (Mot de Passe Visuel Multidimensionnel)**, 
une invention visant à augmenter la sécurité des mots de passe par une représentation 
visuelle multidimensionnelle comprenant :

- Mélange aléatoire des caractères (shuffle)
- Espacement aléatoire
- Couleurs aléatoires par caractère
- Micro-inclinaison (rotation légère)
- Déformation ovale aléatoire
- Micro-vibrations invisibles (anti-OCR)
- Régénération automatique toutes les X minutes

## Usage
Ouvrir `index.html` dans un navigateur moderne (Firefox recommandé).

## Avertissement
Ce prototype illustre le fonctionnement visuel du MPVM. La version finale destinée à 
l'intégration doit inclure :
- Validation serveur des espaces
- Gestion cryptographique interne
- Sécurisation du transport et des accès
